# automacao_python
Projeto 01 do curso OficinaPython.

__Situação:__
Cliente tem uma planilha de possiveis nomes para o seu site e gostaria que você pesquisa-se todos no site REGISTRO.BR para saber se estão disponíveis para uso.
Ele quer o resultado em um arquivo de texto (.txt) para analisar os que estão disponíveis.

__Objetivo__:
A lista de domínios pode conter inúmeros itens e pesquisar de um em um é inviável.
O objetivo é automatizar o processo para pesquisar e para gerar o arquivo de texto.
